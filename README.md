# CrazyCity
 
## Presentation

A dodge game inspired by Mr.Oops!! mobile game and made during GTMK2020 with theme "Out of control" (not submitted)

Gather traffic cone through the city and avoid out of control drivers !


## Controls

P1 / P2 : Arrows / WASD

P: Pause
